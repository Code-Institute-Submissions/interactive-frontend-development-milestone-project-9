How to play

1. Start the game by turning the game on using the ON/OFF button.
2. Press Start to begin playing. Simon will begin by giving a signal, your job is to repeat with the same signal by pressing the right color.
3. Simon will repeat the first signal and add one more for each round. Now repeat in the same order.
4. You win the game by playing all 20 rounds of the game.
5. Difficulty of the game can be set by playing in Strict mode where you only have one chance before you have to start over from the beginning.
